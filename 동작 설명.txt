동작 설명

시작 후 버튼의 랙, 파레트를 누르면 미리보기로 어떤 디자인인지 볼 수 있습니다.
그 후 랙은 원하는 크기를 입력 하고 생성을 누르면 마우스에 반투명 오브젝트가 따라와 바닥을 누르면 생성이 가능합니다
(오른쪽 마우스 클릭, 키보드 S 버튼, 바닥 밖 영역, 생성을 누르면 생성 해제가 됩니다.)

랙, 파레트를 클릭을 하면 빨간 하이라이트가 들어오며 이는 드래그가 가능한 상태입니다.

박스 추가 버튼을 누르면 랙은 칸마다 파레트는 크기에 맞게 박스가 생성됩니다.

랙, 파레트, 영역선은 랙 삭제 키보드 delete, back space 버튼을 누르면 삭제가 됩니다.

바닥 커스텀 확장으로 원하는 방향으로 늘리거나 줄일 수 있습니다.(랙, 오브젝트가 있으면 바닥 축소 금지)

읽기모드 ON을 하면 랙, 파레트를 이동할 수 없으며 하이라이트 색이 빨강 > 초록으로 바뀝니다(박스 하이라이트, 카메라 이동 기능 추가)

영역선 그리기를 누르면 바닥에 노란 선을 마우스 클릭으로 그릴수 있으며 첫 점과 맨 끝점이 이어지면 하나의 객체로 만들어지며
계속해서 생성이 가능합니다.
(키보드 S 버튼, 바닥 밖 영역, 영역선 버튼을 클릭하면 생성 해제가 됩니다.)

카메라 보기 전환을 누르시면 위, 정면, 뒤쪽, 왼쪽, 오른쪽 카메라 전환이 가능하며 되돌리기를 누르면 이동 전 복귀 시점으로 돌아옵니다.

저장/불러오기에서는 랙, 파레트, 영역선, 박스, 바닥의 크기 위치를 저장 후 불러오기가 됩니다.

랙, 파레트의 이동 기능에서는 r키를 누르면 회전, 이동 시 그리드 칸 위 드랍이 있으며 바닥 밖 영역으로 나갈 수 없도록 했습니다.

랙의 단독 기능으로써는 쌓기가 됩니다.(면적이 맞는것들만 정확하게 쌓기가 되며 생성 시에도 쌓기로 미리보기가 올라갑니다.)

랙, 파레트의 크기를 방향키와 ( [ , ] ) 버튼으로 조절할 수 있으며 파레트는 위, 아래 방향키 기능을 제거하여 높이를 조절 할 수 없습니다.

오브젝트 선택시 선택이 안된 오브젝트를 클릭하거나 바닥을 클릭하면 선택이 해제가 되도록 했습니다.

ctrl 드래그로 다중 선택이 되며 각자 쌓기와 이동이 가능합니다.

ctrl C + V 로 복사 붙여넣기가 가능합니다 붙여넣을 기준은 잡은 영역 왼쪽 하단 끝지점입니다.
(붙여넣기 시 오브젝트 끼리 영역과 면적이 맞다면 쌓인채로 생성됩니다)

렌더 표시는 랙, 파레트, 영역선의 갯수와 박스 갯수, draw call 의 갯수를 볼 수 있습니다.
